# 商业化方案

> **更新:** 2026-05-19  
> **适用说明:** Phase 3 启动，Phase 1-2 为商业化铺路

---

## 1. 商业模式

### 1.1 核心模型

```
用户 → 订阅付费 → 你统一调用 LLM API → AI 写日记
     ↗             ↓
  免费额度      成本由订阅覆盖
```

### 1.2 为什么选订阅制

| 理由 | 说明 |
|------|------|
| **可预测收入** | 每月固定，方便规划成本 |
| **降低用户门槛** | 用户不需要折腾 API Key 和充值 |
| **边际成本低** | LLM 每次调用成本很低（<$0.001/篇） |
| **生态健康** | 付费用户 = 真实用户，关注留存而不是拉新 |

## 3. 成本分析

### 3.1 单位经济 (假设 DeepSeek Flash)

| 项目 | 成本 |
|------|:----:|
| AI 写一篇日记 (avg 500 tokens) | ~$0.0003 (0.02 分钱) |
| 图片存储 (每张 1MB，100 张) | ~$0.001 (Cloudflare R2) |
| 服务器 (Linode $10/mo) | 支持 ~500 活跃用户 |
| 每个免费用户月成本 | ~$0.月成本** | ~$0.01 (30 篇 × $0.0003) |
| 每个付费用户月成本 | $0.03-0.10 |
| **每用户毛利** | **基础版 ¥18.90 / 用户** |

### 3.2 盈亏平衡点

```
月度固定成本:
  - 服务器:  ¥70  ($10)
  - 域名:     ¥8   ($1.2)
  - 合计:     ¥78

每用户毛利:
  - 基础版  ¥19 - ¥0.5(LLM+存储) = ¥18.5
  - 高级版  ¥39 - ¥1.0(LLM+存储) = ¥38.0

盈亏平衡点 (假设 60% 基础 / 40% 高级):
  = ¥78 / (¥18.5×0.6 + ¥38×0.4) = ¥78 / ¥26.3
  ≈ 3 个付费用户

只需要 3 个付费用户就能覆盖成本!
```

## 4. 支付接入

### 4.1 &darr; GRD支付选择</s> 支付选择

| 方案 | 优势 | 缺点 | 建议 |
|:----:|:----:|:----:|:----:|
| **Stripe** | 国际标准，API 好，支持支付宝 | 国内用户少 | ✅ (首推，接支付宝) |
| **支付宝当面付** | 国内方便 | 需要个体工商户 | ✅ (用户多选) |
| **Paddle** | 含税务合规 | 手续费稍高 | 备选 |
| **Lemon Squeezy** | 简单，中国用户友好 | 新平台 | 可尝试 |

### 4.2 支付流程

```
用户点击「升级到基础版」
    │
    ▼
  Stripe Checkout 页面
    │
    ▼
  支付完成 → Webhook 通知
    │
    ▼
  更新数据库 (订阅状态 + 额度)
    │
    ▼
  用户解锁全部功能
    │
    ▼
  (每月到期自动扣款)
```

## 5. 从自带 Key 到订阅的迁移

### 5.1 过渡方案

| 时间 | 模式 |
|:----:|:----:|
| **Phase 1-2** | 用户自带 API Key，免费使用 |
| **Phase 3 启动** | 同时支持两种模式：自带 Key (免费) + 订阅 (你买单) |
| **Phase 3+** | 订阅模式为主，自带 Key 作为「免费+高级」选项保留 |

### 5.2 数据迁移

```
用户在 Phase 1 写的日记 → 全量保留
用户购买订阅后 → 切换到你的 API Key，体验无缝
用户想取消 → 数据可导出，也可切回自带 Key 模式
```

### 5.3 用户心理过渡

```
Phase 1: 「这工具真好用，还免费（用我的 Key）」→ 依赖形成
Phase 3: 「不错，¥19 不用折腾 Key，省事」 → 付费意愿
          「S3 加密存我自己那放心」 → 高级版
```

## 7. 法律与合规

### 7.1 需要准备的文档

| 文档 | 说明 |
|------|------|
| **隐私政策** | 说明用户数据如何存储和保护 |
| **服务条款** | 明确使用规则和责任 |
| **退款政策** | 订阅制退款规则 |

### 7.2 数据合规

- 用户数据存储在国内或用户选择的区域
- 阿里云 OSS / 腾讯云 COS 作为国内 S3 备选
- 符合《个人信息保护法》要求
- 用户可随时删除全部数据
