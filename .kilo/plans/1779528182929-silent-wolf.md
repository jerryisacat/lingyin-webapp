# Issue #18 — 📊 数据统计: 首页仪表盘方案

## 方向变更

**不做独立 `/stats` 页面**，改为在登录后首页（`/`）的已登录仪表盘区域增加统计模块。

## 核心思路

| 决策 | 选择 |
|------|------|
| 位置 | 首页仪表盘，保持在 `src/app/page.tsx`（server component）内 |
| 统计视图 | 客户端组件 `<DashboardStats />`，通过 `useEffect` 调用 `/api/stats` |
| 图表方案 | **纯 Tailwind div 柱状图**（不引入 recharts/chart.js 等依赖） |
| 导航 | 不变 — 统计就在首页，无需加导航入口 |
| 现有元素 | 保留问候语、CTA 按钮、小贴士卡片，统计模块插在 CTAs 和小贴士之间 |

## 数据流

```
page.tsx (server)          — auth() 验证会话、渲染 greeting + CTA + stats 模块 + tip
  └─ DashboardStats (client) — useEffect 调用 /api/stats → jsonOk(stats) → render
```

## 后端细节 — `src/lib/stats.ts`

```typescript
import { prisma } from "@/lib/db"
import type { StatsData } from "@/types"

export async function getStats(userId: string): Promise<StatsData> {
  // 1. 批量查询：取所有 entry 的 date/wordCount/tags/imageCount
  const entries = await prisma.entry.findMany({
    where: { userId },
    select: { date: true, wordCount: true, tags: true, imageCount: true },
    orderBy: { date: "desc" },
  })

  // 2. 无数据 → 返回零值
  if (entries.length === 0) { ... }

  // 3. 聚合计算
  const totalWords = entries.reduce((s, e) => s + e.wordCount, 0)
  const totalImages = entries.reduce((s, e) => s + e.imageCount, 0)
  const totalDays = new Set(entries.map(e => dateToStr(e.date))).size

  // 4. Streak：排序 desc → 从今天/昨天开始倒推连续天数
  const streak = calcStreak(entries.map(e => e.date))

  // 5. 月度数据：group by year-month
  const monthlyData = groupByMonth(entries)

  // 6. 标签频率：parse JSON → count → sort → top 20
  const topTags = calcTopTags(entries)

  return { totalWords, totalDays, currentStreak: streak, totalImages, monthlyData, topTags }
}
```

### Streak 算法
```
dates = entries.map(e => dateToStr(e.date)), 降序
ref = today 或 yesterday（取最近一次）
loop:
  如果 ref 在 dates 中 → streak++，ref -= 1 day
  否则 → break
return streak
```

### 标签聚合
```
tags = entries.flatMap(e => JSON.parse(e.tags ?? "[]"))
count = Map<tag, count>()
sort desc → slice(0, 20)
```

### DashboardStats 组件 — 状态处理

```
loading  → skeleton animation (shimmer placeholder 4 cards)
error    → "加载失败" + 重试按钮 (同样处理 chart & tags)
empty    → "还没有日记数据" (仅在 totalDays === 0 时显示)
data     → 正常渲染
```

loading 时将 4 张卡片 + 柱状图 + 标签区域全部展示为灰色 shimmer，避免布局抖动。
